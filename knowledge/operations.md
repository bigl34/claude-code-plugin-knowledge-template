## Returns Process

1. **Return requested** - Customer initiates return within 30 days
2. **RMA issued** - Return authorization provided
3. **Item shipped** - Customer ships item back
4. **Inspection** - Item inspected on receipt
5. **Refund processed** - Refund or exchange completed

### Return Policies
- 30-day return window from delivery
- Items must be unused and in original packaging
- Customer pays return shipping unless item is defective
